# Random User Generator

A simple and responsive web application that fetches and displays random user profiles using the Random User API. The project demonstrates how to work with APIs using JavaScript and dynamically update the user interface with real-time data.

---

## Features

* Fetch random user data from an API
* Display user profile picture
* Show name, phone number, gender, age, and location
* Responsive and clean user interface
* Dynamic content rendering using JavaScript
* Lightweight and easy to understand

---

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Random User API

---

## How It Works

1. Open the application.
2. Click the **Generate User** button (or load the page if users are fetched automatically).
3. The application sends a request to the Random User API.
4. The API returns a randomly generated user profile.
5. The user information is displayed on the page.

---

## API Used

This project uses the **Random User API** to retrieve randomly generated user data.

API Endpoint:

```text
https://randomuser.me/api/
```
